# Stress-Strain Analyzer

A Python tool for calculating and analyzing stress-strain behavior from material test data.

## Features
- Calculates stress and strain values
- Uses sample material test data
- Organized function-based structure
- Separate material data viewer
- Clear printed output

## Technologies Used
- Python
- CSV file handling
- Basic mechanics of materials calculations

## How to Run
Run the script in Python:

python stress_strain_analyzer.py

To view the sample material data:

python data_viewer.py
